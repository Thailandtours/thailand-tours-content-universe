# Eksempler Før og Efter

## Hotelåbning

Før:

```markdown
Hotellet tilbyder moderne faciliteter og en god beliggenhed tæt på stranden.
```

Efter:

```markdown
Stranden ligger ikke som en fjern idé her. Den er dagens naturlige retning, fra morgenkaffen til den sidste tur tilbage mod værelset.
```

Hvorfor:

- Fjerner `hotellet tilbyder`.
- Gør beliggenheden konkret.
- Sætter en scene i stedet for at liste faciliteter.

## Destination

Før:

```markdown
Koh Lanta er en populær destination i Krabi-provinsen med smukke strande og mange aktiviteter.
```

Efter:

```markdown
Koh Lanta har ingen travl skyline. Øen folder sig langs vestkysten med lange strande, små strandbyer og aftener, hvor tempoet falder før solen gør.
```

Hvorfor:

- Starter med karakter, ikke geografi.
- Giver rytme og stedfornemmelse.
- Undgår `populær destination`.

## Landetekst om indrejse

Før:

```markdown
Der kan gælde visumregler, og du bør undersøge dem inden afrejse.
```

Efter:

```markdown
Danske statsborgere skal som udgangspunkt have styr på fire ting før en rejse til Bali: pas, visum eller e-VOA, Bali Tourist Levy og elektroniske ankomst- eller toldformularer. Reglerne kan ændre sig, så tjek altid de officielle sider tæt på afrejse.
```

Hvorfor:

- Fortæller kunden præcis hvad der skal tjekkes.
- Sætter kravene i en overskuelig rækkefølge.
- Beholder forbeholdet uden at blive utydelig.

## Hvem passer stedet til

Før:

```markdown
Området passer til både par og familier.
```

Efter:

```markdown
Par får mest ud af området, hvis ro, god mad og korte aftenture betyder mere end natteliv. Familier trives især her, når strand, pool og nem logistik tæller højere end store forlystelser.
```

Hvorfor:

- Forklarer hvorfor, ikke bare hvem.
- Hjælper kunden med at spejle egne behov.

## Praktisk klimaafsnit

Før:

```markdown
Her er tropisk klima året rundt, og vejret er godt i højsæsonen.
```

Efter:

```markdown
November til marts er typisk det mest stabile vindue, især hvis strand og bådture fylder meget i rejsen. Fra maj til oktober kan regn, vind og bølger påvirke udflugter, så her planlægger vi mere fleksibelt.
```

Hvorfor:

- Undgår vejrløfter.
- Forklarer konsekvensen for rejsen.
- Giver rådgivning uden garanti.
